UT6 - Plantilla DevTools + Git

Completa cada sección mientras aplicas los cambios en tu portfolio.

  • DevTools HTML/CSS
  • DevTools JS
  • Git básico
  • Reto integrador

DevTools: HTML y CSS

1. Indica qué botón/elemento ajustaste.
2. Escribe el cambio exacto (color, padding, sombra, etc.).
3. Pega el CSS definitivo en el área de notas.

DevTools: JavaScript

1. ¿En qué función colocaste `console.log`?
2. ¿Qué mensaje de error apareció al romper el código?
3. ¿Dónde pusiste el breakpoint y qué valor observaste?

Git básico

Registra aquí los pasos:
  • `git init` (sí/no, fecha)
  • `git add ...` (archivos añadidos)
  • Mensajes de commit escritos

Reto integrador

  1. Describe la mejora visual.
  2. Describe la mejora funcional.
  3. Anota los 3 commits finales (style / feat/fix / chore).
