# Better Comments — Proyecto Flask Básico

Usamos etiquetas en comentarios para aprender más rápido:

- # * Idea clave: lo imprescindible que debes recordar
- # ! Advertencia: seguridad o malas prácticas que evitar
- # ? Explicación: contexto o por qué se hace algo
- # TODO: Siguiente mejora a intentar tú mismo

Dónde ver ejemplos:
- `miniapp/__init__.py`: arranque de la app, configuración mínima y DB
- `miniapp/models.py`: qué representa el MODELO
- `miniapp/views.py`: rol del CONTROLADOR (rutas)
- `miniapp/templates/*.html`: comentarios Jinja con {# ... #}
