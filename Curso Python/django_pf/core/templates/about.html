📚 Acerca de esta aplicación Django

🎯 Objetivo

Aprender Django desde cero mediante ejemplos prácticos en un proyecto pequeño. Cada archivo está comentado línea a línea para que entiendas qué hace cada cosa.

⚡ Características del proyecto
  • 📁 Plantillas: Guardadas en core/templates con herencia (base.html → páginas)
  • 🔗 Rutas con parámetros: Tipado automático (<int:id>, <str:nombre>)
  • 📝 Formularios POST: request.POST.get("campo") para recibir datos de formularios
  • 🔍 Query strings GET: request.GET.get("q") para parámetros en la URL (?q=valor)
  • 📡 API JSON: Endpoints que devuelven JSON con JsonResponse()
  • ❌ Manejadores de errores: error_404, error_500 personalizados
  • 🔧 Middleware: RequestLogMiddleware para logs y cabeceras personalizadas
🗂️ Estructura del código
django_pf/
├── manage.py              # 🎮 "Control remoto" del proyecto
├── djpf/                  # 📦 Configuración del proyecto
│   ├── settings.py        # ⚙️ Configuración general (BD, apps, middleware)
│   ├── urls.py            # 🗺️ Rutas raíz (incluye core.urls)
│   ├── wsgi.py            # 🌐 Punto de entrada para producción (sync)
│   └── asgi.py            # 🚀 Punto de entrada para producción (async)
└── core/                  # 📱 Nuestra app principal
    ├── apps.py            # 📋 Configuración de la app
    ├── urls.py            # 🔗 Rutas de la app (/saluda/, /api/...)
    ├── views.py           # 🧠 Lógica de las vistas (HTML + JSON)
    ├── middleware.py      # 🔧 Middleware personalizado (logs, cabeceras)
    └── templates/         # 🎨 Plantillas HTML
        ├── base.html      # 📐 Layout base (navbar, footer)
        ├── index.html     # 🏠 Página de inicio
        ├── about.html     # 📚 Esta página
        └── resultado.html # ✅ Respuesta del formulario
🚀 Próximos pasos para aprender más
  1. Lee el código comentado: Abre core/views.py y lee los comentarios paso a paso
  2. Modifica las rutas: Añade nuevas rutas en core/urls.py y crea vistas
  3. Crea nuevas páginas: Añade templates en core/templates/
  4. Aprende modelos: Crea modelos en core/models.py para guardar datos en BD
  5. Panel de admin: Ejecuta python manage.py createsuperuser y accede a /admin/
  6. Despliega: Sube tu proyecto a Render, Railway o PythonAnywhere
